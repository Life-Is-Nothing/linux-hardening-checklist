# ✅ Linux Hardening Checklist

CIS-inspired **Linux hardening checklist** with automation scripts.

## Contents

- Checklist items for server baseline security
- Scripts under `scripts/` for common hardening tasks

## Quick start

```bash
git clone https://github.com/Life-Is-Nothing/linux-hardening-checklist.git
cd linux-hardening-checklist
ls scripts/
# review then run only on systems you own/manage
```
